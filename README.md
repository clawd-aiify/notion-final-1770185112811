# notion-final-1770185112811

Auto-generated API Sandbox - Mock API with MongoDB persistence.

## Quick Start

```bash
# Install
npm install

# Create .env.local
echo "MONGODB_URI=<your-mongodb-uri>" > .env.local

# Run
npm run dev
```

Visit http://localhost:3000

## API Endpoints

View the home page dashboard for available endpoints and request logging.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- MongoDB

## Deployment

### Vercel

```bash
vercel --prod
```

Set `MONGODB_URI` environment variable in Vercel dashboard.
